
# E-commerce Page

An e-commerce page created with React, Redux and Bootstrap using the fake store API (https://fakestoreapi.com/).


## Technologies Used

-HTML

-CSS

-Bootstrap

-TypeScript

-React JS

-Redux Toolkit




## Features

- Live preview
- Responsive Design


## Run Locally

Clone the project

```bash
  git clone https://github.com/zeus97/ecommerce-concept.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
